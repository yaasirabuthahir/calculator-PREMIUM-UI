# calculator-PREMIUM-UI
A premium neon-style calculator with keyboard support, sound effects, and immersive space-themed UI (stars, shooting stars, and animated intro).

## ✨ Features

- 🔢 Basic calculator operations
- ⌨️ Full keyboard support
- 🔊 Button click sound effects
- 🌙 Space UI (stars, galaxy fog)
- 💫 Shooting stars animation
- 🎬 Logo intro with sound
- 💡 Neon glow button effects
- ⚡ Smooth animations and transitions

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism + Animations)
- JavaScript (Vanilla JS)
